# Lead Management

## Business Goal

Manage potential customers before they become customers.

## Backend Requirements

### Entity
Lead

Fields:
- Id
- FirstName
- LastName
- CompanyName
- Email
- Phone
- Status
- OwnerId
- CreatedDate

### APIs

- Create Lead
- Update Lead
- Get Lead By Id
- Search Leads
- Delete Lead

### Validation

- FirstName required
- CompanyName required
- Email required

## Frontend Requirements

### Pages

#### Lead List

Features:
- Search
- Add Lead
- Edit Lead
- Delete Lead

Grid Columns:
- Name
- Company
- Email
- Status

#### Lead Form

Fields:
- First Name
- Last Name
- Company
- Email
- Phone
- Status

## Acceptance Criteria

- User can create lead
- User can edit lead
- User can search lead
